Promise & Observable

A Promise handles a single event when an async operation completes or fails.

An Observable is like a Stream (in many languages) and allows to pass zero or more events
where the callback is called for each event.


Promise can be used by DOT THEN method
Observable uses .subscribe method


Promise will be asynchronous only
Observable can be synchronous or asynchronous


Promise can not be cancelled
Observables can be cancelled by unsubscribing .
