So whenever we want to share the data in components for that there are 4 methods which are

parent to Child: via @Input

Child to Parent: via @Output() and EventEmitter
by using a event emitter and

Child to Parent: via @ViewChild

Unrelated Components: via a Service
We can also use this service if there is no relation between the components.
inside the service we can use a subject for behaviour from RXJS library.
