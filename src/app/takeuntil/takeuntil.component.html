TakeUntil in RXJX

The takeUntil operator returns an Observable that emits value from the source Observable until
the notifier Observable(provided observable) emits a value.
When the notifier emits a value, the TakeUntil completes the Source observable.
It also monitors a second Observable, notifier that you provide. If the notifier emits a value,
the output Observable stops mirroring the source Observable and completes.
